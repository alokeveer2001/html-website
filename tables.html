HTML Tables

  • Table tag <table>

    The <table> tag is used to create a table in HTML. It serves as a container for all the table elements, including rows and cells.

    Example:

    <table>
    <!-- Table content goes here -->
</table>

  • Table header <thead>

    The <thead> tag is used to group the header content in a table. It typically contains the column headers, which provide context for the data in the table.

    Example:

    <table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
</table>

  • Table body <tbody>

    The <tbody> tag is used to group the main content of a table. It contains the actual data rows of the table.

    Example:

    <table>
    <tbody>
        <!-- Table content goes here -->
    </tbody>
<table>

  • Table row <tr>

    The <tr> tag is used to define a row in a table. Each row contains one or more table cells, which can be either header cells (<th>) or data cells (<td>).

    Example:

    <table>
    <tbody>
        <tr>
            <!-- Table content goes here -->
        </tr>
    </tbody>
<table>

  • Table footer <tfoot>

    The <tfoot> tag is used to group the footer content in a table. It typically contains summary information or totals for the columns in the table.

    Example:

    <table>
    <tfoot>
        <!-- Table content goes here -->
    </tfoot>
</table>

  • Table data <td>

    The <td> tag is used to define a standard data cell in a table. It contains the actual data for the corresponding row and column.

    Example:

    <table>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
    </tbody>
<table>

  • Complete Table Example

    Here is a complete example of an HTML table using all the tags mentioned above:

    <table border = "1">
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
            <th>Header 3</th>
        <tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
            <td>Data 3</td>
        <tr>
        <tr>
            <td>Data 4</td>
            <td>Data 5</td>
            <td>Data 6</td>
        <tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Footer 1</td>
            <td>Footer 2</td>
            <td>Footer 3</td>
        <tr>
    </tfoot>
</table>


    And the table above looks like this:

    Header 1 Header 2 Header 3
    Data 1 Data 2 Data 3
    Data 4 Data 5 Data 6
    Footer 1 Footer 2 Footer 3
