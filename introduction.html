Introduction to HTML

  • What is HTML ?

    HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

    • HyperText: This refers to the ability to create links that connect web pages to one another, making the web a connected "web" of information.
    • Markup Language: This means you use "tags" to surround your content, giving that content meaning and structure. You are "marking up" a plain text document.

    First Website: https://info.cern.ch/hypertext/WWW/TheProject.html

  • The Heading tag (<h></h>)

    Headings are tags used to define titles and subtitles on your page. They are crucial for creating a logical hierarchy and outline for your content.

    There are six levels of headings in HTML, ranging from <h1> to <h6>. <h1> is the most important heading, typically used for the main title of the page, while <h6> is the least important.

    Example:

    Code : 
    <h1>This is a main heading</h1>
<h2>This is a subheading</h2>
<h3>This is a smaller subheading</h3>
    Output :

    This is a main heading

    This is a subheading

    This is a smaller subheading

  • The Paragraph tag (<p></p>)

    The <p> tag is used to define a paragraph of text. It helps to separate blocks of text, making the content more readable and organized.

    Code : 
    <p>This is a paragraph of text.</p>
<p>This is another paragraph of text.</p>
    Output :

    This is a paragraph of text.

    This is another paragraph of text.

  • HTML Horizontal Rules

    The <hr> tag is used to create a horizontal line (or "rule") across the page. It is often used to separate sections of content visually.

    Example:

    <hr>

  • The Line Break Tag (<br>)

    The <br> tag is used to insert a line break within a block of text. It is useful for breaking lines without starting a new paragraph.

    Code : 
    This is the first line.<br>This is the second line.
    Output :
    This is the first line.
    This is the second line.

  • The Anchor Tag (<a></a>)

    The <a> tag is used to create hyperlinks, which allow users to click and navigate to other web pages or resources.

    The <a> tag uses the "href" attribute to specify the destination URL of the link.

    Example:

    Code : 
    <a href="https://www.example.com">This is a link to Example.com</a>
    Output :
    This is a link to Example.com

  • HTML boilerplate code

    Boilerplate code refers to the standard structure and elements that are commonly included in an HTML document. This code provides a basic framework for your webpage and ensures that it adheres to web standards.

    Example of a basic HTML boilerplate:

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name = "viewport" content = width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
<!-- Your content goes here -->
</body>
</html>

    Explanation of the boiler plate

    • <!DOCTYPE html>: This declaration defines the document type and version of HTML being used (HTML5 in this case).
    • <html lang="en">: This is the root element of the HTML document. The "lang" attribute specifies the language of the document (English in this case).
    • <head>: This section contains meta-information about the document, such as its character encoding, viewport settings for responsive design, and the title that appears in the browser tab.
    • <meta charset="UTF-8">: This tag sets the character encoding for the document to UTF-8, which supports a wide range of characters from different languages.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: This tag helps ensure that the webpage is responsive and displays correctly on different devices by setting the viewport width to match the device's width and setting the initial zoom level to 1.0.
    • <title>Document Title</title>: This tag defines the title of the document, which appears in the browser tab and is important for SEO.
    • <body>: This section contains the main content of the webpage that will be displayed to users. You can add text, images, links, and other elements here.
