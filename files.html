HTML file paths

  • Relative file paths

    Relative file paths specify the location of a file in relation to the current document. They are useful for linking to files within the same project or directory structure.

    Example:

    <a href="images/photo.jpg">Link to a photo</a>

    In this example, "images/photo.jpg" is a relative path that points to a file named "photo.jpg" located inside an "images" folder, which is in the same directory as the current HTML file.

  • Absolute file paths

    Absolute file paths specify the complete location of a file on a computer or server. They start from the root directory and include all directories leading to the file.

    Example in Windows:

    <a href="C:\Users\Username\Documents\project\index.html">Link to index.html</a>

    Example in MacOS/Linux:

    <a href="/Users/Username/Documents/project/index.html">Link to index.html</a>

    In these examples, the absolute path points to the "index.html" file located in the "project" folder, which is inside the "Documents" directory of the user "Username".
