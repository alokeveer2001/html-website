HTML Lists

What are Lists in HTML?

In HTML, lists are used to group related items together. They make information easier to read and organize. Lists can be ordered (numbered), unordered (bulleted), or descriptive (term-definition).

Types of Lists in HTML

1. Unordered Lists (<ul></ul>)

  • Displays items with bullets by default.
  • Best for lists where the order doesn't matter.
Example Code Example Output 
                            
    <ul>
        <li>Coffee</li>
        <li>Tea</li>
        <li>Milk</li>
    </ul>
  • Coffee
  • Tea
  • Milk

2. Ordered Lists (<ol></ol>)

  • Displays items with numbers or letters.
  • Best for step-by-step instructions or rankings.
Example Code Example Output 
                            
    <ol>
        <li>Wake up</li>
        <li>Brush teeth</li>
        <li>Eat breakfast</li>
    </ol>
  1. Wake up
  2. Brush teeth
  3. Eat breakfast

3. Description Lists (<dl></dl>)

  • Used to define terms and their descriptions.
  • Contains:
    • <dt></dt> : Definition term
    • <dd></dd> : Definition description
Example Code Example Output 
                            
    <dl>
       <dt>HTML</dt>
       <dd>- A markup language for creating webpages.lt;/dd>
       <dt>CSS</dt>
       <dd>- A stylesheet language for stylingwebpages.</dd>
       <dt>JavaScript</dt>
       <dd>- A programming language for addinginteractivity.</dd>
    </dl>
HTML
- A markup language for creating webpages.
CSS
- A stylesheet language for styling webpages.
JavaScript
- A programming language for adding interactivity.

Nested Lists

  • Lists can be placed inside other lists to create subcategories.
  • Useful for menus, outlines, and structured data.
Example Code Example Output 
                            
    <ul>
        <li>
            Subjects
            <ol>
                <li>
                    Science
                    <ul>
                        <li>Physics</li>
                        <li>Chemistry</li>
                        <li>Biology</li>
                    </ul>
                </li>
                <li>Mathematics</li>
                <li>History</li>
            </ol>
        </li>
    </ul>
  • Subjects
    1. Science
      • Physics
      • Chemistry
      • Biology
    2. Mathematics
    3. History
